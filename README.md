## Description

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## API Documentation

# List all messages
GET localhost:3000/messages
# Create a new message
POST localhost:3000/messages
content-type: application/json
{
    "content": "okokok"
}
# Get a particular message
GET localhost:3000/message/123456
